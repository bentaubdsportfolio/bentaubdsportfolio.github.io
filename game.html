Game Design

This is a presentation of a game design document I made for a Video Game Design course.

The idea behind this assignment was to take an existing movie/TV franchise, try to create a game based around it, and then pitch it to our professor to see if he would greenlight it's development. I took the animated series, Gurren Lagann, and turned it into a "Beat em' up" style game, taking inspiration from several games in the same genre.
